CSS

Sintaxi del CSS

Els elements del css van de la següent manera: "p(que seria l'element o elements afectats){propietat: valor de la propietat;}"

Un element de css pot tenir infinites declaracions

Selectors

En el CSS hi ha 3 tipus de selectors:

  • Segons element: es posa el tipus d'element al que volem afectar: p{}
  • Segons class: es posa quan volem afectar a una class concreta: .guapos{}
  • Segons ID: volem afectar a un sol element que té una id única: #potaDreta{}

També podem fer que un conjunt de declaracions css afectin a classes, elements i ids diferents(com un || al javascript), mira l'exemple: div, .tontos, #potaEsquerre {color: blue;}

Podem ser més específics i fer que certes declaracions afectin només a certs tipus d'elements que tinguin certes classes(com un && al javascript), mira l'exemple: p.llestos{font-size:14px;};

Altre manera de ser específic és afectar a elements només quan estant dins d'altres elements específics: div > p{}. Això només afectaria a elements p que es trobin directament dins d'un div

Podem fer també que l'element sigui afectat quan es trobi dins d'un altre independentment del nivell, mira l'exemple: article p{}

Aquestes dues maneres anteriors d'afectar elements específics també poden funcionar amb classes

PseudoSelectors

Hi ha selectors del css que només funcionen en certes circumstàncies, són els anomenats pseudoSelectors i alguns d'aquests són

  • :link afecta a enllaços no visitats
  • :visited afecta a enllaços que ja han sigut visitats
  • :hover afecta a l'element quan el ratolí està a sobre sense clicar
  • :active afecta a l'element quan és clicat(en el moment del clic i es manté si el clic és mantingut)
  • :nth-child(...) entre parèntesi hem de posar un número, aquest serà la posició de l'element afectat(per exemple, div:nth-child(4) {} afectaria al quart div de la pàgina)

Emplaçament del css

Existexen 3 maneres d'aplicar el css:

  • Dins de l'element que volem afectar, com a un atribut style que tingui com a valor la propietat css que volem
  • Com a un element style dins de l'element header, amb totes les propietats que busquem
  • En un arxiu extern posant l'element link dins de head fen una cosa així: <link rel="stylesheet" type="text/css" href="style.css">. Aquesta és la manera més utilitzada, ja que és la més còmode i ho podem aplicar a diversos html alhora

Resolució de conflictes

Hi ha vegades que es pot aplicar dues propietats contradictories que afecten a un element, la resolució de conflictes seria el fer quina propietat queda aplicada sobre l'element, hi ha diverses maneres de fer-ho

  • Origen: l'última declaració guanya, tenint en compte que el html s'executa de dalt a baix, guanya la que està més a baix
  • Herència: les propietats que es defineixin sobre un element son heretades pels elements dins del primer element. Les propietats definides sobre subelements(un p dins un div, per exemple) guanyen sobre propietats en nivells més alts
  • Especificitat: la propietat més específica guanya. L'especificitat segueix aquest ordre:
    1. Propietat definida dins el propi element
    2. Propietat que té ID com a target
    3. Propietat que té com a target una classe, pseudo-classe o atribut
    4. Propietat que afecta a un nom d'element

Per acabar, si poses !important al costat d'una propietat, faràs que aquesta propietat tingui més pes que qualsevol altre, però no ho fagis servir gaire que és una merda de mantenir

Maneres d'estilar text

Aquí mirarem maneres de decorar i estilar text

  • font-family: escull la tipografia del text. Exemple: Times New Roman
  • color: el color del text. Exemple: red, #0066ee
  • font-style: estil del text(cursiva, negreta i tal). Exemple: italic
  • font-weight: nivell de negreta. Exemple: 200, 700, bold
  • font-size: tamany de la lletra. Exemple: 22px, 120%, 2em(aquest em multiplica pel número el tamany que tenia la lletra per defecte)
  • text-transform: Transformació de la lletra. Exemple: capitalize, lowercase
  • text-align: alineació del text. Exemple: justify, center, left

Les caixes

Les caixes son la manifestació dels elements del html, les més vistes són les div. Aquestes caixes podem modificar-les pel css

Maneres de modificar-les serien:

  • background-color: Modifica el color de fons de l'interior de la caixa
  • padding: és l'espai que separa el contingut de la caixa del seu marge
  • border: el marge de la caixa
  • margin: és l'espai que separa el marge de la caixa amb la resta de contingut de fora
  • width: amplada del contingut de la caixa
  • height: alçada del contingut de la caixa. tant en width com en height podem establir un min i un max si volem que el tamany sigui dinàmic
  • box-sizing: "border-box". Fem que width i height tingui en compte el padding i border. Aquesta propietat no s'hereda, si volem que sigui aplicat a tots els elements hem de posar la propietat a l'element *{}
  • overflow: ajusta el contingut segons el valor, mira els possibles
    • visible: fa que el contingut sobresurti
    • hidden: amaga el contingut que no cap
    • auto: afegeix una barra de despaçament a la dreta, no es prefereix
    • scroll: afegeix barres de desplaçament tant verticals com horitzontals, aquest és el pitjor

Propietat background

La propietat background seria el que té de fons un element o la plana en si. Podem posar com a fons un color o una imatge

En cas que sigui una imatge, per defecte aquesta es repetirà per a omplir l'element del qual és fons, però podem canviar la manera en la que actua amb la propietat background-repeat

Totes les característiques del background es poden definir dins de la propietat background

Posicionat per floating

El posicionat per floating és la manera més utilitzada per a posicionar elements. Un cop apliquem un float a un element, passarà a només interactuar amb elements float(dins dels límits d'altres elements que el continguin)

Amb float hem de fer que un element floti cap a la dreta o cap a l'esquerra. Amb clear fas ue un element pugui interactuar amb elements float, però fas que aquest element que té clear no tingui altres elements float a la dreta, esquerre o els dos amb both

Posicionat per relative i absolute

Abans de parlar del relative o l'absolute, hem de parlar del posicionat estàtic, que és el que hi ha per defecte per tots els elements excepte el html

El relative seria com un posicionament float però pot interactuar amb els elements que no són float. Aquest element conserva l'espai ocupat de la seva posició original

L'absolute és semblant al relative, però aquest depèn completament de l'element que el conté i en l'ambient de la plana, es mou amb l'element en el que resideix. Per a que absolute fagi cas, hem de fer que l'element que el conté tingui una posició absolute o relative

Media queries

Les media queries són les maneres d'adaptar l'estil depenent del tamany de la pantalla

Es defineixen amb @media(condició que aplica si és true){}. Normalmet es posa com a condició max-width i min-width

Podem aplicarmés d'una condició posant 'and' o ','(la , seria un or), separant cada condició dins del seu propi parèntesi

Procura no crear conflictes entre condicions. Normalment es fa 3 espais de condicions per tamany:

  • Gran: (min-width: 1200px)
  • Mitjà: (min-width: 992px) and (max-width: 1199px)
  • Petit: (max-width: 991px)

Disseny responsiu

El disseny responsiu és l'idea d'aplicar diferents tamanys en els elements depenent del tamany de la pantalla. Normalment es fa imaginant la plana en 12 columnes i fer que els elements s'adaptin al tamany d'aquestes

Una manera de fer-ho, és fer classes predeterminades al css i després posar els elements necessaris a la classe depenent necessitat, un exemple d'això ho veuràs més tard amb el bootstrap

Recorda que per a fer que els telèfons mòbils s'adaptin al disseny responsiu sense tocar els pebrots, hem de posar el metatag <meta name="viewport" content="width=device-width, initial-scale=1">

Bootstrap

El bootstrap és una llibreria d'arxius que es pot descarregar gratuïtament. Conté arxius css i javascript i serverix per a assistir a la web que es pretengui crear; tenen, entre d'altres, classes predeterminades de les que hem parlat abans al disseny responsiu, com les 'col-md-4', 'col-lg-2'...

Els arxius bootstrap es poden descarregar aquí: Bootstrap

Per a fer servir el sistema de cel·les del bootstrap, el que hem vist abans de les 12 columnes, hem de definir un element amb la class container, després un element a dins amb la class row i finalment a dins ja podem definir un element amb la class de divisió per columnes que ens interessa
