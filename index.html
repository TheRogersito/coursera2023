CSS

Sintaxi del CSS

Els elements del css van de la següent manera: "p(que seria l'element o elements afectats){propietat: valor de la propietat;}"

Un element de css pot tenir infinites declaracions

Selectors

En el CSS hi ha 3 tipus de selectors:

  • Segons element: es posa el tipus d'element al que volem afectar: p{}
  • Segons class: es posa quan volem afectar a una class concreta: .guapos{}
  • Segons ID: volem afectar a un sol element que té una id única: #potaDreta{}

També podem fer que un conjunt de declaracions css afectin a classes, elements i ids diferents(com un || al javascript), mira l'exemple: div, .tontos, #potaEsquerre {color: blue;}

Podem ser més específics i fer que certes declaracions afectin només a certs tipus d'elements que tinguin certes classes(com un && al javascript), mira l'exemple: p.llestos{font-size:14px;};

Altre manera de ser específic és afectar a elements només quan estant dins d'altres elements específics: div > p{}. Això només afectaria a elements p que es trobin directament dins d'un div

Podem fer també que l'element sigui afectat quan es trobi dins d'un altre independentment del nivell, mira l'exemple: article p{}

Aquestes dues maneres anteriors d'afectar elements específics també poden funcionar amb classes

PseudoSelectors

Hi ha selectors del css que només funcionen en certes circumstàncies, són els anomenats pseudoSelectors i alguns d'aquests són

  • :link afecta a enllaços no visitats
  • :visited afecta a enllaços que ja han sigut visitats
  • :hover afecta a l'element quan el ratolí està a sobre sense clicar
  • :active afecta a l'element quan és clicat(en el moment del clic i es manté si el clic és mantingut)
  • :nth-child(...) entre parèntesi hem de posar un número, aquest serà la posició de l'element afectat(per exemple, div:nth-child(4) {} afectaria al quart div de la pàgina)
